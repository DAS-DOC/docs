## Overview of Payment Options API

The Payment Options API is a comprehensive payment gateway solution that allows you to accept various payment methods across different platforms, ensuring a seamless and secure payment experience for your customers. Our API offers:

<CardGroup cols={2}>
  <Card
    title="Multiple Payment Methods"
    icon="pen-to-square"
  >
    Including card payments, Pay By Link, QR code payments, digital wallets (Apple Pay, Google Pay), subscriptions, and more
  </Card>
  <Card
    title="Scalability"
    icon="image"
  >
    Our API is designed to handle large volumes of transactions, making it suitable for businesses of any size.
  </Card>
   <Card
    title="Security"
    icon="pen-to-square"
  >
    Built-in 3D Secure (3DS) compliance and adherence to PCI DSS 4.0 standards.
  </Card>
  <Card
    title="Developer-Friendly"
    icon="image"
  >
    Easy-to-use API references, Postman collections, and code samples to speed up integration.
  </Card>
</CardGroup>

## Overview of Payment Options API

The Payment Options API is a comprehensive payment gateway solution that allows you to accept various payment methods across different platforms, ensuring a seamless and secure payment experience for your customers.

### Key Features

* **Multiple Payment Methods**: Including card payments, Pay By Link, QR code payments, digital wallets (Apple Pay, Google Pay) and subscriptions.

* **Scalability**: Our API is designed to handle large volumes of transactions, making it suitable for businesses of any size.

* **Security**: Built-in 3D Secure (3DS) compliance and adherence to PCI DSS 4.0 standards.

* **Developer-Friendly**: Easy-to-use API references, Postman collections, and code samples to speed up integration.

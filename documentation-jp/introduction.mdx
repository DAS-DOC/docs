# ファイルのダウンロードリンクの作成方法

ドキュメント内でファイルをダウンロードできるリンクを作成する方法には、いくつかの方法があります：

## 基本的なダウンロードリンク

最も簡単な方法は、HTMLの`a`タグに`download`属性を使用する方法です：

```html
<a href="/path/to/your/file.pdf" download>ファイルをダウンロード</a>
```

## Markdownでのリンク作成

Markdownを使用している場合は、以下のように記述できます：

```markdown
[ファイルをダウンロード](/path/to/your/file.pdf)
```

## 注意点

- ファイルパスは、プロジェクトのルートディレクトリからの相対パスを使用します
- 大きなファイルの場合は、ファイルサイズを明記することをお勧めします
- サポートされているファイル形式を確認してください

## 例

```markdown
# ダウンロード可能なファイル

- [サンプルPDF (2.5MB)](/samples/example.pdf)
- [テンプレートファイル](/templates/template.xlsx)
- [ソースコード (ZIP)](/downloads/source-code.zip)
```

これらのリンクをクリックすると、ブラウザはファイルのダウンロードを開始します。

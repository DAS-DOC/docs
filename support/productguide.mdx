### **Download Individual Product Guides**

- **[Pay by Link User Guide (PDF)](./path/to/PayByLink_User_Guide.pdf)**  
   Set up and manage payment links that can be sent to customers for seamless, one-click payments.

- **[QR Code Payment User Guide (PDF)](./path/to/QRCode_User_Guide.pdf)**  
   Instructions for generating and utilizing QR codes linked to a Hosted Payment Page for in-person and online transactions.

- **[Subscription Service User Guide (PDF)](./path/to/Subscription_User_Guide.pdf)**  
   Manage recurring payments with flexible billing cycles and subscriber management options.

- **[Scheduler User Guide (PDF)](./path/to/Scheduler_User_Guide.pdf)**  
   Set up and use Scheduler to manage appointments and bookings linked with payment processing.

- **[Hosted Payment Page (HPP) User Guide (PDF)](./path/to/HPP_User_Guide.pdf)**  
   Customize and use the Hosted Payment Page to provide a secure checkout experience for customers.
